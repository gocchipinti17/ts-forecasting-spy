# ts-forecasting-spy

Time Series Forecasting: An Exploratory Study Using S&P 500 ETF Data

This project is an exercising studying the applications of time series forecasting techniques to predict financial stock prices. Investors looking to maximize profit from financial investing can benefit from data-driven approaches to estimating future stock prices and forecasting techniques can be a part of such approaches. This project particularly uses 10-year monthly price data to forecast the SPY-USA ETF price using the following time series forecasting models: exponential smoothing (ETS), naïve, seasonal naïve, and an ensemble method.
